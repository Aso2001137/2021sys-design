```uml
@startuml
[*] -> 紹介ページ
紹介ページ -> ログイン : アカウントをお持ちの方ボタンを押す
ログイン -> 掲示板 : ログインボタンを押す
紹介ページ -> 新規登録 : 初めての方ボタンを押す
新規登録 ->　掲示板 : 登録ボタンを押す
掲示板 -> 他プロフ : 掲示板書込み者の名前を押す
他プロフ -> 掲示板 : 掲示板に戻るボタンを押す
他プロフ -> チャット : チャットボタンを押す
他プロフ ->　依頼 : 依頼ボタンを押す
依頼 -> 支払い : 支払いボタンを押す
支払い -> 依頼確定 : 確定ボタンを押す
依頼確定 ->　掲示板 : 掲示板に戻るボタンを押す
掲示板 -> 自プロフ : 自分の名前を押す
自プロフ -> 掲示板 : 掲示板に戻るボタンを押す
自プロフ -> 依頼品確認 : 依頼品を確認するボタンを押す
@enduml
```
